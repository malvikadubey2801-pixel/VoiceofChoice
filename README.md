# Voice of Choice – AI Emotional Support Assistant

## Overview
Voice of Choice is a prototype built during Ideatex Ideathon (DevBraze, REVA University). It provides basic emotional support based on user stress input.

## Features
- Stress level input using heart rate slider
- Categorization (exam, health, relationships, general)
- Context-based supportive responses

## Tech Stack
- HTML/CSS/JS (or whatever you used)

## Note
This is a 1-day ideathon prototype. The full vision includes biological signal-based stress detection and personalized voice responses.
